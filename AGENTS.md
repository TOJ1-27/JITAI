# Project Rules

- The website default language is English. Chinese content is available only through the language toggle.
- Do not mix English and Chinese copy within the same page language state.
- Do not invent company data, certifications, capacity claims, technical parameters, customers, or test results.
- Do not commit secrets, passwords, tokens, credentials, or real `.env` files.
- Do not remove the video Hero, RFQ form, or valid links unless a task explicitly requires it.
- After page changes, run lint, format check, and production build.
- Check the site at 390px, 768px, 1024px, and 1440px.
- Keep GitHub Pages `/JITAI/` subpath compatibility.
- At the end of each task, report changed files, check results, risks, and TODOs.
